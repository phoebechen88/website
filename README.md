# Portfolio

This repository is for a personal portfolio built with React + Vite (JavaScript). 


Files of interest

- `index.html` — app root
- `src/main.jsx` — entry
- `src/App.jsx` — routes and layout
- `src/pages/*` — Home, About, Projects, Contact
- `src/components/*` — Header, Footer, ProjectCard, ProjectGrid
- `src/data/projects.js` — projects
- `public/resume.pdf` — resume PDF
